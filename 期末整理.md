## 領導
1. 領導能力
    + 職位權力
        + 獎勵、懲罰、法定
    + 個人權力
        + 專業能力、資訊與關係、個人魅力
2. 領導特質
    + 行動能力、高度自信、創新能力、專業能力
    + 激勵能力、保持彈性、誠信正義
3. 感知與行為
    + 感知(Preception) 過程
    + 勞資心理契約
        + 資方期待
        + 勞方期待
    + 態度
        + 認知(Cognition) 結果
        + 情緒
        + 行為/動機
4. 自我效能
    + 正向經驗、楷模學習、讚美激勵、情緒鼓勵
5. 情境式領導
    + 指導型、授權型、參與型、說服型
6. 溝通
    + 溝通模型、有效傾聽、評論處理、(建設性意見
7. 談判
    + 談判目標
        + 結果達成、維持關係
    + 有效的談判
        + 雙方接受
        + 談判效率
        + 和諧關係
8. 衝突
    + 定義與種類
        + 實質衝突
        + 情緒衝突
    + 衝突管理風格
        + 迴避、接受、競爭、協同、合作
9. MBTI 性格分析
    + 人際關係：內向、外向
    + 資料搜集：細節、整體
    + 決策行為：理性、感性
    + 工作模式：規律、彈性
